# GitHub Public Surface Crawl

Git-governed control plane for the approved **GitHub Public Knowledge Surface Crawl — Plan v0.2**.

This repository is the public crawler contact surface and the source of truth for plans, amendments, policies, run manifests, inventories, validation receipts, and closeout records. The initial migration preserves the approved plan only; it does not authorize reconnaissance, crawling, ledger design, executor provisioning, or a crawl run.
